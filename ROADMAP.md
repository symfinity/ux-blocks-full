## Details

### 0.1.x — Initial metapackage

Five-tier catalog, Symfony Flex recipe, optional vertical suggestions, installation handbook, verification guide

### 0.2.x — Installation refinement

Improved installation flow, refined verification, clearer Stage B guidance, package consistency

### 0.3.x — Catalog growth

Support for newly released official UX Blocks packages, updated recommendations, expanded package coverage

### 0.4.x — Vertical packages

Official marketing package, official ecommerce package, refined Composer suggestions

### 0.5.x — Bundle lifecycle

Bundle registration refinements, dependency alignment, lifecycle consistency across all tiers

### 0.6.x — Documentation

Expanded handbook, installation scenarios, upgrade guidance, ecosystem documentation

### 0.7.x — Developer experience

Improved onboarding, diagnostics, installation validation, smoother first-run experience

### 0.8.x — Ecosystem alignment

Alignment with Flex recipes, Chameleon ecosystem, Symfony releases, package compatibility

### 0.9.x — Release preparation

Dependency freeze, documentation review, migration notes, release candidate preparation

### 1.0.x — Stable catalog

SemVer-stable metapackage, stable package selection, long-term maintained official UX Blocks catalog
